Этот проект предназначен для парсинга текста с новостного ресурса.
Программа инициирует создание папок в директории проекта соответствующих числу страниц сайт, заданных пользователем.
В созданных папках формируются файлы в формате .txt, соответсвующие искомому типу новостей, введенных пользователем.
Результатом работы программы является набор файлов в формате .txt с именами,
соответсвующими названиям статей, распределенные по папкам с соответсвующими номерами страниц сайта.
