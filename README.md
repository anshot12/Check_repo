# Конвертор валют

Программа осуществяет конвертацию валюты по курсу из данных сайта.

Курсы валюты относительно доллара и евро загружается автоматически, посолькую являются наиболее актуальными пр  конвертации.

Результатом работы программы является вывод конвертированной суммы из исходной валюты в искомую по актуальному курсу на данный момент.
