# Check_repo
Данная программа осуществляет сбор данных с назначенного сайта.
Пользовательский ввод состоит из: 1. кол-ва страниц для осуществления поиска по сайту, 2. наименованию интересующего типа статей
Результатом работы программы является формирования набора папок в репозитории проекта, с сответсвующими названиями
Содержимое папок представляет собой набор файлов в формате .txt с соответсвующими наименованиями статей, искомого типа
