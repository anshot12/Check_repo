# Генератор текста
Данная программа осуществляет генерацию текста на основе триграммов из заданного файла обучения.

Программа разбивает текста на пары триграммов (head (1 слово), tail (2 слова)) создавая список.

На основе данного списка формируется словарь, на основе ключа head со значением списка из встрчающихся tail во всем тексте.

При помощи функции random.choices на основе весовых характеристик встрчающихся в тесте совпадений триграммов генерируется текста состоящий из 10 предожений, длинной не менее 5 слов. Предложения формируются из head начинающихся с заглавной буквы.

Результатом работы программы является генерация текста из  10 предложений на основе предоставленного текствового файла, для создания триграммов.
