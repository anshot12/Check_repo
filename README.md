# Карточки
Программа представляет собой игру.

Пользователю предоставляется приложение для изучения любых терминов состоящих из: названия и определения

В программе реализованно логирование данных пользовательского ввода в файл .txt в репозитории программы.

Программа имеет функции импорта данных, эспорта, добавления, удаления карточе а также сбора статистики ответов пользователя.
